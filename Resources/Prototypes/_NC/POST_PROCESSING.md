# Модуль: Визуальная атмосфера и Постобработка (Post-Processing)

## 1. Цель
Трансформация стандартного визуала в мрачный, высококонтрастный ретрофутуризм Найт-Сити (Cyberpunk 2045). Система имитирует восприятие мира через цифровые оптические импланты и кинематографическую пленку.

## 2. Техническая архитектура
Модуль реализован через `PostProcessOverlay.cs`, работающий в `OverlaySpace.WorldSpace`. Это гарантирует, что эффекты применяются только к игровому миру, не затрагивая пользовательский интерфейс (UI).

### Конвейер рендеринга (Render Pipeline):
1. **Threshold Pass:** Выделение ярких участков экрана в уменьшенную текстуру.
2. **Blur Pass:** Размытие ярких участков (Bloom) через `IClyde.BlurRenderTarget`.
3. **Composite Pass:** Финальный шейдер (`post_process.swsl`), объединяющий все эффекты.

## 3. Основные эффекты

### 3.1. ACES Filmic Tonemapping
Использование эталонной кривой ACES для перевода HDR-цветов в SDR.
- **Эффект:** Мягкое "сгорание" яркого света и глубокие, насыщенные тени. Убирает "пластиковый" вид пиксель-арта.

### 3.2. Cinematic Color Grading (Teal & Orange)
Раздельная тонировка яркостных диапазонов.
- **Тени (Luma < 0.8):** Глубокий бирюзово-синий подтон (`rgb(0.02, 0.04, 0.1)`).
- **Свет (Luma > 0.8):** Усиление насыщенности (Neon Vibrance x1.3).
- **Защита черного:** Математический фильтр предотвращает окрашивание абсолютно черных участков, сохраняя глубину контраста и предотвращая "синий туман".

### 3.3. High-Res Bloom
- **Порог (Threshold):** 0.5 (захватывает насыщенный неон).
- **Интенсивность:** Регулируется через настройки (дефолт 150%).
- **Реализация:** Аддитивное смешивание размытой маски яркости.

### 3.4. Dynamic Vignette
Радиальное затемнение краев экрана.
- **Параметры:** Радиус 0.8, Мягкость 0.5.
- **Цвет:** Настраиваемый (по умолчанию черный).

### 3.5. Film Grain (Зернистость)
Анимированный процедурный шум.
- **Анимация:** Привязана к `TIME`, создавая эффект "кипящей" матрицы импланта.
- **Прозрачность:** Регулируется через настройки (дефолт 60%).

## 4. Настройки (CVars)
Все параметры доступны в `Escape -> Options -> Graphics`:

| CVar | Описание | Значение по умолчанию |
|------|----------|-----------------------|
| `graphics.bloom` | Включение свечения | `true` |
| `graphics.bloom_intensity` | Сила свечения (0-200) | `150` |
| `graphics.vignette` | Включение виньетки | `true` |
| `graphics.film_grain` | Включение шума | `true` |
| `graphics.film_grain_strength` | Интенсивность шума | `60` |

## 6. Рекомендации по настройке
Для изменения атмосферы (например, при переходе в другую зону):
1. Изменяйте `shadowColor` в `post_process.swsl` для смены настроения (например, на багровый для Пустошей).
2. Используйте `tintWeight` для контроля силы тонировки в зависимости от игровых событий.
